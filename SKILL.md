# Stock Learning Skill

为A股小白投资者生成每日学习日报，用最通俗的语言解释复杂的股市知识。

## 每日推送时间
- **早上 6:00** — 前一日市场回顾 + 当日开盘前瞻
- **晚上 18:00** — 当日市场收盘分析 + 明日展望

## 日报结构

### 1. 今日市场速览
抓取东方财富、同花顺等平台的A股市场数据，生成简洁的涨跌概览。

信息来源：
- 东方财富 A股行情：`https://quote.eastmoney.com`
- 同花顺市场概况：`https://www.10jqka.com.cn/`
- 新浪财经 A股：`https://finance.sina.com.cn/stock/`

### 2. 今日主题（深度分析）
选取当日热门行业或话题，用小白能懂的语言深度拆解。

选股逻辑（优先级从高到低）：
1. 当日涨幅超3%的热门板块
2. 政策利好的行业（央行、证监会最新政策）
3. 机构最新研报推荐方向
4. 长期价值投资标的（蓝筹、白马）

信息来源：
- 东吴证券周报：`https://www.dwz.cn/f4wujie`（搜索）
- 招商证券研报
- 平安证券周报
- 远瞻慧库行业分析

### 3. 投资观点
给出2-3个近期值得关注的方向，注明是短线还是长线。

格式：
- 🎯 **短线机会**：[名称] — 逻辑简述
- 🌿 **长线布局**：[名称] — 逻辑简述

### 4. 小白课堂
每期讲解一个基础概念，用比喻+实例的方式让零基础也能听懂。

参考：references/beginner-lessons.md

## 输出格式规范

```
📊 A股学习日报 | {日期}

【今日市场速览】
- 三大指数：上证 {点位}（{涨跌%}）、深证 {点位}（{涨跌%}）、创业板 {点位}（{涨跌%}）
- 成交额：两市合计 {金额}
- 热门板块：{板块1}、{板块2}、{板块3}
- 北向资金：{净流入/出} {金额}

【今日主题】{主题名称}
{300字左右的深度分析，用通俗语言，适合小白理解}

【投资观点】
🎯 短线：{股票/行业名称} — {1句话逻辑}
🌿 长线：{股票/行业名称} — {1句话逻辑}

【小白课堂】{知识点名称}
{200字通俗讲解，包含生活中的比喻帮助理解}
```

## 内容获取流程

1. **市场数据**：搜索"东方财富 A股 {日期}"获取当日行情
2. **热门板块**：搜索"A股 热门板块 {日期}"获取当日涨幅前排
3. **研报解读**：搜索相关行业研报，用web_fetch抓取摘要
4. **政策信息**：搜索"证监会/央行 最新政策 {日期}"获取政策面信息

## 注意事项

- 所有分析仅供学习参考，不构成投资建议
- 优先选择有明确数据来源的信息
- 用词通俗易懂，避免专业术语，必要时附上简单解释
- 长线/短线须明确标注
