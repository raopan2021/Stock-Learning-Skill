# 第十八课：选股思路

A股有5000多只股票，如何选出值得买的？这一课介绍三种选股思路。

## 一、三大选股维度

| 维度 | 关注点 | 适合人群 |
|------|--------|---------|
| 消息面 | 政策、公告、研报 | 短线操作 |
| 技术面 | K线、均线、成交量 | 趋势交易 |
| 基本面 | 业绩、估值、行业 | 中长线投资 |

> 💡 **建议新手以基本面为主，技术面为辅**。

## 二、消息面选股

### 1. 政策利好

国家政策扶持 = 行业机会。

| 政策信号 | 可能受益板块 |
|---------|------------|
| 新能源补贴 | 整车、电池 |
| 芯片国产替代 | 半导体设备 |
| 医改 | 创新药 |
| 房地产松绑 | 地产链 |

### 2. 公告挖掘

上市公司公告包含重要信息：

| 公告类型 | 信息含义 |
|---------|---------|
| 业绩预增 | 净利润大幅增长，可能是利好 |
| 重大合同 | 有了大客户，业绩有保障 |
| 高送转 | 股票拆分，对资金面有影响 |
| 减持公告 | 股东在卖，小心 |

### 3. 研报追踪

机构研报透露专业判断：

- **首次覆盖**：机构第一次研究，可能看好
- **目标价**：机构认为能涨到多少
- **评级上调**：机构更看好了

## 三、技术面选股

### 1. 趋势选股

顺势而为，只做上涨趋势的股票：

- 股价在20日均线上方
- 均线多头排列
- 成交量配合放大

### 2. 突破选股

在股价突破重要位置时买入：

- 突破前期高点
- 突破震荡区间
- 突破均线压力

### 3. 超跌选股

跌多了可能反弹（高风险）：

- 股价跌破净资产
- 连续下跌后出现底部信号
- 机构大比例净买入

## 四、基本面选股

这是**最重要的选股维度**，也是价值投资的核心。

### 1. 好行业

先选赛道，再选公司：

| 好的行业特点 | 举例 |
|------------|------|
| 需求稳定 | 医药、消费 |
| 行业集中度提升 | 行业龙头越做越大 |
| 政策支持 | 新能源、芯片 |

### 2. 好公司

用财务指标筛选：

| 指标 | 筛选标准 |
|------|---------|
| ROE | >15% |
| 净利润增速 | >10% |
| 资产负债率 | <60% |
| 经营现金流 | 为正且稳定 |

### 3. 好价格

好公司也要有好价格：

- PE在历史低位
- PB在历史低位
- 股息率高于银行理财

## 五、三种选股策略对比

| 策略 | 优点 | 缺点 |
|------|------|------|
| 消息面 | 快速，效率高 | 容易滞后，庄家骗线 |
| 技术面 | 有明确买卖点 | 滞后性，有假信号 |
| 基本面 | 稳健，中长期有效 | 需要研究能力，考验耐心 |

> 💡 **建议**：用基本面选股，用技术面择时。

## 六、新手选股建议

1. **从熟悉的行业开始**：你用啥产品，就关注啥公司
2. **先看龙头**：行业里最大、最赚钱的公司往往最稳
3. **不要追题材**：小白追题材，十追九套
4. **控制数量**：持仓不要超过3-5只

## 思考题

> 你看好新能源行业，想要找一只新能源股票。你会从哪些维度来筛选？
