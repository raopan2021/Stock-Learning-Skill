# 第十七课：止损与止盈

**会买是徒弟，会卖才是师傅。** 这一课讲如何体面地离场。

## 一、为什么要止损？

> **止损不是为了赚钱，而是为了控制亏损。**

### 数字告诉你止损的重要性

| 亏损幅度 | 回本需要涨多少 |
|---------|--------------|
| 10% | 11% |
| 20% | 25% |
| 30% | 43% |
| 50% | 100% |
| 80% | 400% |

> 💡 亏得越多，回本越难。**及时止损是保护本金最有效的方式。**

## 二、止损的方法

### 方法1：固定比例止损

亏到某个比例就卖，比如**-8%**。

> 10元买的，8元止损（亏20%就卖）

### 方法2：技术位止损

跌破重要支撑位就卖。

> 股价跌破20日均线，止损

### 方法3：逻辑止损

买入的理由消失了，就卖。

> 买的时候看好业绩增长，结果业绩暴雷，马上卖

## 三、止损的常见误区

> ⚠️ **这些止损方式是错的！**

| 误区 | 问题 |
|------|------|
| 跌了不卖，等反弹 | 小亏变大亏 |
| 跌了就补仓，越跌越买 | 可能越陷越深 |
| 止损设太宽 | 亏太多，难以回本 |
| 止损设太窄 | 被市场噪音频繁止损 |

## 四、什么是止盈？

**止盈 = 赚钱了，卖不卖？**

涨的时候要学会"落袋为安"，不然只是"纸上富贵"。

## 五、止盈的方法

### 方法1：目标收益率止盈

达到目标收益就卖：

| 投资类型 | 建议目标 |
|---------|---------|
| 短线 | 10-15% |
| 中线 | 20-30% |
| 长线 | 50%+ |

### 方法2：分批止盈

涨到目标后，分批卖出：

- 第一次止盈：卖1/3
- 第二次止盈：再卖1/3
- 剩下的继续持有

### 方法3：高处回落止盈

从最高点回落**15-20%**就清仓。

> 💡 适合：**不知道涨到哪里是顶**的情况。

## 六、止损 vs 不止损

### 不止损的人在想什么？

- "再等等，肯定会反弹"
- "卖了就真亏了"
- "我不卖，就不算亏"

> 🚨 **这是散户最常见的错误**，鸵鸟心态只会让亏损越来越大。

## 七、止损/止盈纪律

> 记住：**制定计划，严格执行**。

```
买入前想好：
- 止损位在哪？（比如-8%）
- 目标位在哪？（比如+20%）
- 什么情况下提前止盈？
```

到了位置就执行，不要临时改计划。

## 思考题

> 你10元买了某股票，现在涨到13元（+30%）。你会止盈吗？如果继续涨到15元呢？
