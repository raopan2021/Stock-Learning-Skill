# 第九课：K线基础

K线是技术分析的基本功。这一课让你从零看懂K线图。

## 一、K线是什么？

K线（蜡烛图）是一种**记录股价走势的图形**，每个K线代表一段时间：

- **日K**：每天一根
- **周K**：每周一根
- **月K**：每月一根

## 二、K线的构成

```
     最高价
       │
  开盘 │ 收盘
  ─────┤     ← 实体
       │
     最低价
```

每个K线有**4个价格**：开盘价、收盘价、最高价、最低价

## 三、红涨绿跌（A股规则）

| 类型 | 颜色 | 条件 |
|------|------|------|
| 阳线 | 红色（涨） | 收盘价 > 开盘价 |
| 阴线 | 绿色（跌） | 收盘价 < 开盘价 |

> 💡 注意：有些软件用红涨绿跌，有些相反，要注意区分。

## 四、单根K线的含义

### 1. 大阳线（长红实体）

**形态**：实体很长，几乎没有上下影线

**含义**：买盘强劲，多方完全主导。大涨信号。

### 2. 大阴线（长绿实体）

**形态**：实体很长，几乎没有上下影线

**含义**：卖盘强劲，空方完全主导。大跌信号。

### 3. 十字星

**形态**：开盘=收盘，上下影线差不多长

**含义**：多空博弈平衡，是**转折信号**，但方向不确定。

### 4. 上影线很长

**形态**：上方影线很长，实体小

**含义**：冲高回落，上方卖压大。**偏弱信号**。

### 5. 下影线很长

**形态**：下方影线很长，实体小

**含义**：探底回升，下方支撑强。**偏强信号**。

## 五、K线组合（入门级）

### 1. 吞没形态

**看涨吞没**：绿色小实体 + 红色大实体完全包裹

**看跌吞没**：红色小实体 + 绿色大实体完全包裹

### 2. 早晨之星（底部反转）

三根K线组合：
1. 大阴线（跌）
2. 小实体（震荡）
3. 大阳线（涨）

**含义**：行情可能见底回升

### 3. 黄昏之星（顶部反转）

三根K线组合：
1. 大阳线（涨）
2. 小实体（震荡）
3. 大阴线（跌）

**含义**：行情可能见顶回落

## 六、怎么看K线周期？

| 周期 | 用途 |
|------|------|
| 日K | 看短期走势，做短线参考 |
| 周K | 看中期趋势，辅助判断方向 |
| 月K | 看长期趋势，判断大方向 |

> 💡 新手建议主要看**日K和周K**，月K太长，容易看不清细节。

## 七、K线的局限性

> ⚠️ **K线是辅助工具，不是预测神器。**

- K线只能反映**已经发生的事**
- 主力可以"画K线"（故意做出某种形态引诱散户）
- 要结合成交量、均线、消息面一起看

## 思考题

> 某股票连续3天收了上影线很长的K线，这意味着什么？
