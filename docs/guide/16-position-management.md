# 第十六课：仓位管理

仓位管理是**最重要的风险控制手段**，也是区分散户和成熟投资者的关键。

## 一、什么是仓位？

**仓位 = 你买了多少股票/基金 ÷ 你总共有多少钱**

| 仓位状态 | 含义 |
|---------|------|
| 空仓 | 100%现金，0%股票 |
| 轻仓 | 20-30%持股 |
| 半仓 | 50%持股 |
| 重仓 | 70-80%持股 |
| 满仓 | 100%持股 |
| 杠杆 | 借钱炒股（危险！） |

## 二、为什么仓位管理重要？

### 没有仓位管理的后果

> 满仓跌50% → 需要涨100%才能回本

### 有仓位管理的好处

> 50%仓位跌50% → 只亏25%，容易回本

> 💡 **控制仓位，就是在控制风险。**

## 三、仓位管理原则

### 1. 不满仓原则

**永远不要满仓**。预留现金，跌了可以补仓拉低成本。

### 2. 分批建仓原则

| 方式 | 操作 |
|------|------|
| 一次建仓 | 一下子全买进去 |
| 分批建仓 | 先买30%，跌了再买30%，再跌再买40% |

> 建议新手用**分批建仓**，降低买贵的风险。

### 3. 大盘仓位原则

| 大盘状态 | 建议仓位 |
|---------|---------|
| 大盘强势上涨 | 60-80% |
| 大盘震荡 | 30-50% |
| 大盘弱势下跌 | 10-30% |

## 四、新手仓位建议

### 保守型（推荐新手）

| 总资金 | 仓位 | 股票金额 |
|--------|------|---------|
| 10万 | 30% | 3万 |
| 10万 | 预留 | 7万（等待机会） |

### 激进型（有经验）

| 总资金 | 仓位 | 股票金额 |
|--------|------|---------|
| 10万 | 60% | 6万 |
| 10万 | 预留 | 4万（补仓用） |

## 五、仓位管理口诀

```
新手三成仓，不满不空常；
跌了有钱补，涨了不后悔。
```

## 六、补仓的技巧

| 情况 | 补仓建议 |
|------|---------|
| 跌5% | 可以补一次 |
| 跌10% | 可以再补 |
| 跌20% | 谨慎，可能是趋势变了 |
| 跌30% | 先分析原因，不要盲目补 |

> ⚠️ **补仓是双刃剑**：补对了降低成本加快回本，补错了加大亏损。

## 思考题

> 你有10万元本金，现在大盘在低位震荡，你打算怎么分配仓位？说说你的理由。
