# 第二十课：下单与成交

这是入门教程的最后一课，讲解实际买卖股票的操作流程。

## 一、下单方式

### 1. 市价单

以当前市场价格立即成交。

**优点**：保证成交
**缺点**：成交价格不确定（尤其是波动大的时候）

### 2. 限价单

设定一个价格，只有达到这个价格才成交。

**优点**：价格确定
**缺点**：可能成交不了

> 💡 **建议新手用限价单**，特别是A股有涨跌停限制，不会差太多。

## 三、买入实操步骤

以券商APP为例（涨乐财富通）：

1. **打开APP** → 点"交易"
2. **点击"买入"**
3. **输入股票代码**（如600519）
4. **输入买入价格**（限价单需要填写）
5. **输入买入数量**（100股=1手）
6. **确认买入**
7. **输入交易密码**
8. **下单成功**

## 四、卖出实操步骤

1. **点击"持仓"**
2. **选择要卖的股票**
3. **点击"卖出"**
4. **填写卖出价格和数量**
5. **确认卖出**

## 五、成交规则

股票成交遵循两个原则：

| 原则 | 说明 |
|------|------|
| **价格优先** | 买的时候，出价高的先成交；卖的时候，出价低的先成交 |
| **时间优先** | 价格相同时，先挂单的先成交 |

### 涨停板时怎么成交？

如果一只股票涨停了，你想买进：
- 按"价格优先"，你的买入价必须≥涨停价
- 按"时间优先"，你挂单越早，成交机会越大

> 所以涨停板买不进是常态，第二天开盘也不一定涨。

## 六、常见问题处理

| 问题 | 解决方案 |
|------|---------|
| 下单提示"资金不足" | 账户里钱不够，需要银证转账 |
| 下单提示"股票停牌" | 停牌期间无法交易，等复牌 |
| 卖出提示"持仓不足" | 你没持有那么多股票 |
| 成交了但看不到 | 持仓更新有延迟，等1-2分钟 |
| 想撤单怎么撤 | 在"当日委托"里找"撤单" |

## 七、银证转账

买股票之前，钱要从银行卡转到证券账户：

1. 打开券商APP → "交易" → "银证转账"
2. 选择"转入"
3. 输入金额
4. 确认

> 💡 建议：**收盘后把多余的钱转回银行卡**，控制冲动交易。
