# 第五课：板块与行业

大盘指数告诉你整体冷热，但真正能赚钱的是**结构化的机会**——哪些行业在涨，哪些在跌。

## 一、什么是板块？

板块就是**按某种逻辑把股票分组**。

| 分组逻辑 | 例子 |
|---------|------|
| 行业 | 白酒板块、银行板块、半导体板块 |
| 地域 | 上海板块、雄安新区板块 |
| 概念 | 人工智能板块、芯片概念、新能源汽车概念 |

> 💡 板块是动态的，"人工智能"以前不是板块，现在成了最热的概念之一。

## 二、行业分类

A股按行业分为以下大类：

### 十大一级行业

| 行业 | 典型股票 | 特点 |
|------|---------|------|
| 🍺 食品饮料（白酒） | 茅台、五粮液 | 业绩稳定，防御性强 |
| 🏦 银行 | 工商银行、招商银行 | 低估值，高分红 |
| 💊 医药 | 恒瑞医药、药明康德 | 刚需，长期稳健 |
| 🔋 新能源 | 宁德时代、比亚迪 | 高成长，波动大 |
| 💻 半导体 | 中芯国际、寒武纪 | 政策支持，国产替代 |
| 🏗️ 房地产 | 万科、保利 | 周期性强，受政策影响大 |
| ☁️ 互联网 | 腾讯、阿里 | 港股为主，平台经济 |
| 🛢️ 能源 | 中国石油、中国石化 | 周期性强，受大宗商品影响 |
| 🏭 制造业 | 美的、格力、海尔 | 家电龙头，业绩稳定 |
| 🚗 汽车 | 比亚迪、上汽 | 转型新能源 |

## 三、热门板块的形成逻辑

### 1. 政策驱动

国家政策鼓励什么，什么板块就容易涨。

> **举例**：国家大力发展新能源汽车 → 整车、电池、充电桩整个产业链都受益

### 2. 业绩驱动

某行业业绩普遍超预期，资金蜂拥买入。

> **举例**：茅台业绩大增 → 白酒板块整体上涨

### 3. 题材驱动

市场喜欢炒概念，跟风资金涌入。

> **举例**：ChatGPT火 → 人工智能概念股连续涨停

> ⚠️ **题材炒作风险极大**，来得快去得也快，新手谨慎追题材。

## 四、如何发现热门板块？

### 方法1：看涨幅榜

每天收盘后，在券商APP里看**行业板块涨幅排名**，涨幅最大的就是当天热门。

### 方法2：看资金流向

资金往哪个板块流，哪个板块就可能有机会。

> 东方财富APP：点"数据中心" → "行业资金流向"

### 方法3：看新闻

政策利好、行业景气度提升，都可能引爆板块。

> 建议关注：央视新闻、东方财富网、同花顺

## 五、板块轮动规律

不同市场环境下，资金会"接力棒"一样在不同板块之间流动：

### 牛市各阶段

| 牛市阶段 | 表现最好的板块 |
|---------|--------------|
| 初期 | 券商股（"春江水暖鸭先知"） |
| 中期 | 消费、科技轮流表现 |
| 后期 | 银行、地产等蓝筹补涨 |

### 熊市/震荡市

| 市场环境 | 表现最好的板块 |
|---------|--------------|
| 熊市 | 医药、消费（防御性板块） |
| 震荡市 | 题材股、短线机会 |

## 六、什么是"概念题材"？

概念是比行业更细的分类，通常是**一个热点主题**带动的股票集合。

### 常见概念举例

| 概念 | 相关股票 |
|------|---------|
| 人工智能 | 科大讯飞、寒武纪、商汤 |
| 芯片国产替代 | 中芯国际、华大九天 |
| 新能源汽车 | 比亚迪、宁德时代、整车厂 |
| 创新药 | 恒瑞、百济神州 |
| 央企改革 | 中字头央企 |

> ⚠️ 概念炒作是"击鼓传花"，最后一个接盘的人会亏得很惨。

## 七、实操：怎么看板块

打开券商APP（以华泰涨乐财富通为例）：

1. 首页 → 点"行情"
2. 选择"板块" → "行业板块"
3. 按涨幅排序，看今日热门
4. 点进某个板块，看成分股

> 💡 建议每天收盘后花5分钟看一下今日涨幅/跌幅前三的板块，培养市场感觉。
