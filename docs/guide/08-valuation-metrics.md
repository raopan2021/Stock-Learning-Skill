# 第八课：常用估值指标

知道公司盈利了，还要判断"贵不贵"。估值就是给股票称重。

## 一、市盈率（PE）

### 公式

```
PE = 股价 ÷ 每股收益（EPS）
```

### 人话版

**"多少年回本"**。PE=30，就是以现在的利润算，需要30年才能赚回本金。

### 生活比喻

你买了一套房子，每年收租2万，房价100万：
- 回本年限 = 100 ÷ 2 = **50年**
- PE = 50

这意味着你的租金回报率只有2%（1/50）。

### A股PE参考

| 行业 | 合理PE范围 | 原因 |
|------|-----------|------|
| 银行 | 5-8倍 | 业绩稳定，但没有成长性 |
| 消费 | 20-35倍 | 品牌溢价，业绩稳定 |
| 医药 | 30-50倍 | 刚需+成长性 |
| 科技 | 50倍+ | 预期未来高速增长 |
| 钢铁/煤炭 | 5-10倍 | 强周期，盈利不稳定 |

> ⚠️ **PE不能跨行业比较**：银行业PE低不代表便宜，科技PE高不代表贵，因为行业特性完全不同。

### PE的使用陷阱

| 陷阱 | 说明 |
|------|------|
| 低PE陷阱 | 低PE可能是因为公司要走下坡路了 |
| 高PE陷阱 | 高PE可能是市场预期太高，业绩不及就暴跌 |
| 负PE | 公司亏损时PE无效 |

## 二、市净率（PB）

### 公式

```
PB = 股价 ÷ 每股净资产
```

### 人话版

**"每股净资产是股价的多少倍"**。PB=3，就是股价是每股净资产的3倍。

### 什么情况下PB更重要？

当PE失效时（公司亏损），用PB判断：

| 情况 | 用什么估值 |
|------|-----------|
| 盈利稳定 | PE为主 |
| 公司亏损 | PB为主 |
| 金融类（银行/保险） | PB更准确 |
| 轻资产公司（互联网） | PB参考价值低 |

> 💡 **银行股用PB**：每股净资产稳定，PB<1意味着股价跌破净资产，理论上"便宜"。

## 三、股息率

### 公式

```
股息率 = 年度每股分红 ÷ 当前股价 × 100%
```

### 人话版

**"收租回报率"**。你买了一只股票，每年分红能拿多少回报。

### A股高股息率股票

| 股票 | 股息率 | 特点 |
|------|--------|------|
| 工商银行 | ~5% | 国有大行，稳定 |
| 建设银行 | ~5% | 国有大行，稳定 |
| 长江电力 | ~4% | 水电，稳定 |
| 中国神华 | ~8% | 煤炭周期股 |

### 股息率的意义

- 股息率高的股票适合**不想炒短线、追求稳定现金流**的投资者
- 被称为"收息股"或"价值股"

## 四、ROE（净资产收益率）

### 公式

```
ROE = 净利润 ÷ 净资产 × 100%
```

### 人话版

**"公司用股东的钱，创造了多少回报"**。

### 判断标准

| ROE | 评价 |
|-----|------|
| >20% | 非常优秀 |
| 15%-20% | 优秀 |
| 10%-15% | 良好 |
| <10% | 一般 |

> 💡 **巴菲特最爱ROE**：他选股的重要标准就是ROE长期>15%。

## 五、估值总结表

| 指标 | 公式 | 用途 | 适用场景 |
|------|------|------|---------|
| PE | 股价/EPS | 判断贵不贵 | 盈利稳定的公司 |
| PB | 股价/每股净资产 | 判断贵不贵 | 亏损/金融类公司 |
| 股息率 | 分红/股价 | 衡量现金回报 | 追求分红的投资者 |
| ROE | 净利润/净资产 | 衡量盈利能力 | 选好公司 |

## 六、如何综合判断估值？

> 好公司 + 好价格 = 值得买入

**好公司**：ROE>15%，业绩稳定增长
**好价格**：PE在历史低位，或低于行业平均

**举例**：
- 茅台是好公司，但PE长期在30-50倍，不便宜
- 工商银行估值低（PE=5），但成长性差

**所以**：好公司买贵了也可能亏损，差公司买便宜了也可能赚钱。

## 思考题

> 某只股票PE=5（很低），另一只PE=50（很高）。你会选哪只？为什么？
