# 第十课：均线系统

均线是技术分析中最常用的工具，简单但非常实用。

## 一、什么是均线？

**均线 = 过去N天的平均价格连线**

| 均线 | 含义 |
|------|------|
| 5日均线（MA5） | 最近5天平均价格 |
| 10日均线（MA10） | 最近10天平均价格 |
| 20日均线（MA20） | 最近20天平均价格 |
| 60日均线（MA60） | 最近60天平均价格 |

> 💡 均线就是"趋势线"，帮助你看清楚价格到底是往上还是往下。

## 二、均线的核心用法

### 1. 均线多头排列（看涨）

所有均线从上到下依次是：5日 > 10日 > 20日 > 60日

**含义**：短期 > 中期 > 长期，说明股票处于强势上涨趋势。

### 2. 均线空头排列（看跌）

所有均线从上到下依次是：60日 > 20日 > 10日 > 5日

**含义**：短期 < 中期 < 长期，说明股票处于下跌趋势。

### 3. 均线粘合（变盘信号）

多条均线挤在一起，距离很近

**含义**：市场到了选择方向的时候，可能大涨也可能大跌。

## 三、金叉与死叉（最重要的信号）

### 金叉（买入信号）

短期均线从下往上穿过长期均线

**举例**：5日均线上穿10日均线 = 金叉

**含义**：短期趋势转强，可能开启上涨。

### 死叉（卖出信号）

短期均线从上往下穿过长期均线

**举例**：5日均线下穿10日均线 = 死叉

**含义**：短期趋势转弱，可能开始下跌。

> 💡 **金叉买，死叉卖**是最简单的技术操作策略，但不适合单独使用。

## 四、均线的支撑与压力

### 支撑位（撑住不跌）

股价下跌时，碰到均线往往能"撑住"

**原因**：均线代表平均成本，很多人在这个价格买入，亏了不愿意卖，形成支撑。

### 压力位（压住不涨）

股价上涨时，碰到均线往往会"压住"

**原因**：之前在均线附近买入的人解套了，想卖，形成压力。

### 实操技巧

- 股价在均线上方 → 偏强
- 股价在均线下方 → 偏弱
- 股价回踩均线不破 → 可能是买入机会
- 股价反弹到均线附近 → 可能是卖出机会

## 五、葛兰碧八大法则

这是均线最经典的用法，记住4个买点和4个卖点：

### 买点（4个）

1. **均线从下降转平时**，股价上穿均线
2. **股价回踩均线**，不破均线后反弹
3. **股价连续上涨后**，回调到均线附近止跌
4. **股价远离均线**，但均线仍在上行

### 卖点（4个）

1. **均线从上升转平时**，股价下穿均线
2. **股价反弹到均线**，无法突破均线后回落
3. **股价连续下跌后**，反弹到均线附近滞涨
4. **股价远离均线**，但均线已走平或下行

## 六、均线的局限性

> ⚠️ **均线是滞后指标**，因为它用的是过去的数据。

- 均线信号**晚于价格**，往往错过最低点和最高点
- 震荡行情中，均线**频繁交叉**，假信号很多
- 要结合K线形态、成交量、消息面一起用

## 思考题

> 某股票5日均线刚刚上穿20日均线形成金叉，但股价还在60日均线下方。你觉得这个时候应该买入吗？
