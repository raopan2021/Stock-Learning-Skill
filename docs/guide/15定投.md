# 第十五课：基金定投

定投是"懒人理财神器"，特别适合上班族和新手。

## 一、什么是定投？

**定投 = 固定时间 + 固定金额 + 买同一只基金**

比如：每月8号发工资当天，花1000元买沪深300ETF。

## 二、定投的原理

**核心逻辑**：市场低时买得多，市场高时买得少，自动摊平成本。

| 时间 | 股价 | 买入金额 | 买入股数 |
|------|------|---------|---------|
| 第1个月 | 1元 | 1000元 | 1000股 |
| 第2个月 | 0.8元 | 1000元 | 1250股 |
| 第3个月 | 0.5元 | 1000元 | 2000股 |
| 第4个月 | 1元 | 1000元 | 1000股 |

> 平均成本 = 总投入 ÷ 总股数 = 4000 ÷ 5250 ≈ **0.76元**

第4个月涨回1元时，你的收益 = (1-0.76) × 5250 = **1260元**！

## 三、定投为什么能赚钱？

**微笑曲线**：即使你买在高点，只要坚持定投，市场反弹时就能盈利。

```
        高点
       /    \
      /      \
     /   反弹  \
    /__________\  ← 回本+盈利
```

> 💡 定投最怕的是：**在低点停止定投，甚至割肉**。

## 四、适合定投的品种

| 品种 | 特点 | 适合人群 |
|------|------|---------|
| 沪深300ETF | 稳健，长期向上 | 新手首选 |
| 创业板ETF | 波动大，收益高 | 能承受波动的人 |
| 科创50ETF | 高风险高回报 | 年轻人 |
| 指数基金 | 费率低 | 长期投资 |

> 💡 **建议新手从沪深300ETF定投开始**。

## 五、定投的频率

| 频率 | 优点 | 缺点 |
|------|------|------|
| 每周 | 成本更平滑 | 需要设置多次 |
| 每月 | 简单，适合工资族 | 稍粗略 |

> 大多数人选择**每月定投**，和发工资节奏一致。

## 六、定投的误区

> ⚠️ **定投不是稳赚不赔的！**

| 误区 | 真相 |
|------|------|
| "随时买都行" | 高位开始定投，可能长期亏损 |
| "不用管，躺着赚钱" | 要定期检视，必要时止盈 |
| "金额固定不变" | 收入增加了，定投金额也应该增加 |

## 七、如何设置定投？

在券商/银行APP里：

1. 找到你想定投的ETF或基金
2. 点击"定投" → 设置每月几号扣款
3. 设置每期金额
4. 确认

## 八、定投止盈技巧

定投要学会**止盈**，不然坐过山车很痛苦：

| 方法 | 说明 |
|------|------|
| 目标止盈 | 收益率达到30%就卖出一半 |
| 分批止盈 | 涨到目标后，分3批卖出 |
| 高点回落止盈 | 从最高点回落15%就清仓 |
