# 第十二课：价值投资

这一课介绍**巴菲特**的投资理念，这是被证明长期有效的投资方法。

## 一、什么是价值投资？

**核心理念**：买的不是股票代码，而是公司的真实价值。

> 好公司 + 好价格 = 值得买入

价值投资者相信：**股价最终会反映公司的真实价值**，所以要做的就是在价格低于价值时买入，耐心等待。

## 二、股神巴菲特的核心理念

### 1. 买股票就是买公司

股票不是一张炒来炒去的纸，而是公司的一部分。

> 巴菲特从不关心股价的短期波动，他关心的是公司业务是否健康。

### 2. 在别人恐惧时贪婪，在别人贪婪时恐惧

这是巴菲特最著名的一句话。

- 市场大跌时 ≠ 应该抛售，而是寻找被低估的好公司
- 市场大涨时 ≠ 应该追高，而是检查是否有泡沫

### 3. 护城河理论

好公司要有**护城河**（竞争优势），让竞争对手难以超越：

| 护城河类型 | 例子 |
|-----------|------|
| 无形资产（品牌/专利） | 茅台的品牌 |
| 转换成本 | 微信/QQ，你换不掉 |
| 网络效应 | 淘宝，用户越多越值钱 |
| 成本优势 | 比亚迪的电动车成本控制 |

## 三、价值投资三步法

### 第一步：找到好公司

什么样的公司是好公司？

- **好行业**：需求稳定，不容易被颠覆
- **好商业模式**：能持续赚钱，现金流好
- **好管理层**：诚实、有能力

### 第二步：评估内在价值

这家公司真实值多少钱？

方法包括：
- PE、PB等估值指标
- DCF现金流折现（专业方法）
- 净资产附近买入（格雷厄姆方法）

### 第三步：等到好价格

好公司也要有好价格。

> 茅台是好公司，但如果在PE=70倍时买入，可能好几年都不赚钱。

**好价格信号**：
- PE在历史低位
- 熊市整体下跌时
- 公司遇到短期困难但本质没坏

## 五、价值投资的难点

> ⚠️ **价值投资知易行难，主要挑战：**

1. **需要耐心**：可能3-5年才见效，急用钱的人不适合
2. **需要研究能力**：要能判断公司是否真的好
3. **需要逆向思维**：大家都在卖的时候，你敢买吗？
4. **需要承受波动**：即使买对了，短期也可能跌30%
