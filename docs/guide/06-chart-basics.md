# 第六课：盘面基础知识

终于要看"盘"了！这一课教你读懂股票软件里的各种数字和图表。

## 一、分时图

分时图是股票**当天每分钟的走势线**。

### 怎么看

- **黄线**（加权平均线）：当天成交的平均价格
- **白线**（指数线）：大盘/个股的实时走势
- 黄线在白线上方 → 当天小盘股更强
- 白线在黄线上方 → 当天大盘股更强

> 💡 简单记忆：**黄线是人，白线是市**（个股看黄线，大盘看白线）

### 关键点位

| 名称 | 含义 |
|------|------|
| 昨收价 | 昨天收盘的价格 |
| 开盘价 | 今天9:30第一笔成交价格 |
| 最高价 | 当天最高成交价 |
| 最低价 | 当天最低成交价 |
| 现价 | 当前最新成交价 |

## 二、K线图（蜡烛图）

K线图是**记录股价走势最重要的工具**，每个K线代表一段时间（1分钟/日/周/月）。

### K线怎么画

```
       最高价
         │
    开盘价│收盘价
    ─────┤
         │
       最低价
```

**阳线（红色/实心）**：收盘价 > 开盘价 = 涨
**阴线（绿色/空心）**：收盘价 < 开盘价 = 跌

### K线的基本含义

| K线形态 | 含义 |
|--------|------|
| 大阳线（大涨） | 多方强势，买方力量强 |
| 大阴线（大跌） | 空方强势，卖方力量强 |
| 上影线很长 | 冲高回落，上方有压力 |
| 下影线很长 | 探底回升，下方有支撑 |
| 小阳/小阴线 | 多空平衡，观望为主 |

> 💡 **上影线长** = 涨上去被人打下来 = 压力
> **下影线长** = 跌下去被人拉起来 = 支撑

## 三、成交量

成交量就是**当天成交了多少股**，用柱状图显示在K线下方。

### 成交量的意义

| 现象 | 信号 |
|------|------|
| 价涨量增 | 上涨有资金推动，健康 |
| 价涨量缩 | 上涨动力不足，小心 |
| 价跌量增 | 下跌有恐慌盘，谨慎 |
| 价跌量缩 | 下跌动能减弱，可能见底 |

> 💡 **量和价是"形影不离"的好搭档**，单独看价不靠谱，要结合量一起看。

## 四、股票软件常用功能

| 功能 | 作用 |
|------|------|
| F10 | 看公司基本面（财务、股东、高管） |
| 资金流向 | 看大单净流入/出 |
| 筹码分布 | 看持仓成本分布 |
| 公告 | 看公司最新公告 |
| 新闻 | 看公司相关新闻 |

## 五、盘口术语

| 术语 | 含义 |
|------|------|
| 内盘 | 主动性卖出的成交量（绿色） |
| 外盘 | 主动性买入的成交量（红色） |
| 五档 | 买1-5、卖1-5的挂单 |
| 量比 | 今日成交量/过去5日均量，>1放量 |
| 换手率 | 当天成交股数/总流通股 |

> 外盘 > 内盘 = 买的人多 = 偏强

## 六、集合竞价看盘技巧

每天9:15-9:25是集合竞价时间：

1. 看开盘价是否大幅高开/低开（超过±5%要警惕）
2. 看竞价时的成交量（量太大可能是诱多）
3. 9:20前可以撤单，9:20后不能撤

> 🚨 9:15-9:20挂涨停价买入的人经常被"钓鱼"，庄家先拉高引你买入，9:20前撤单。

## 思考题

> 某股票今天收了一根"大阳线"，但成交量却比昨天萎缩了一半。这说明什么？应该怎么看？
