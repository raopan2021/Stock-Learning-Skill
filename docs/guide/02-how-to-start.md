# 第二课：如何迈出第一步？

上一课我们理解了"股票是什么"，这一课讲**实战前的准备**——开户和选择券商。

## 一、开户准备

### 需要准备什么？

| 准备项 | 说明 |
|--------|------|
| 身份证 | 有效期内的居民身份证 |
| 银行卡 | 绑定三方存管，通常支持常见银行 |
| 手机 | 接收验证码、完成身份验证 |
| 网络 | 开户全程线上完成，约10-15分钟 |

> 💡 现在开户都是**全程线上**，不需要去营业部排队了。

## 二、选择券商

国内券商有140多家，主流的选择标准：

### 大券商（推荐新手）

| 券商 | 特点 | 适合人群 |
|------|------|---------|
| **华泰证券** | 用户多，APP好用（涨乐财富通） | 新手友好 |
| **中信证券** | 券商龙头，研究实力强 | 有追求的投资者 |
| **国泰君安** | 老牌大券商 | 稳健型 |
| **招商证券** | 服务好，银行背景 | 追求服务的用户 |
| **银河证券** | 佣金低，线下网点多 | 预算敏感型 |

### 选择券商看什么？

1. **APP体验** — 好不好用，卡不卡顿
2. **佣金高低** — 现在普遍万1~万3（后文详解）
3. **线下网点** — 万一有问题能不能找到人
4. **服务质量** — 研报、客服响应速度

> ⚠️ 不推荐小券商：App容易卡顿、服务不稳定、万一倒闭了麻烦。

## 三、什么是股票交易佣金？

券商收的手续费，就是"佣金"。

### 佣金组成

| 费用项 | 收费标准 | 归谁 |
|--------|---------|------|
| 印花税 | 成交金额的0.05%（卖出时收） | 国家 |
| 过户费 | 成交金额的0.001%（双向） | 交易所 |
| 佣金 | 最高不超过成交金额的0.3%（双向） | 券商 |

### 举例说明

你买了1万元股票（单边），又卖掉了：

| 费用 | 计算 | 金额 |
|------|------|------|
| 印花税 | 10000 × 0.05% | 5元（仅卖时收） |
| 过户费 | 10000 × 0.001% × 2 | 0.2元 |
| 佣金 | 10000 × 0.03% × 2 | 6元（按万3算） |
| **合计** | | **约11.2元** |

> 💡 现在很多券商佣金已经降到**万一**甚至**万0.854**，开户时可以跟客户经理谈。

## 四、三方存管是什么？

就是你把银行卡和证券账户绑定，钱从银行卡进证券账户，安全性由银行保障。

> 简单理解：你老婆（银行）管钱，你（券商）管交易，分工明确，互相监督。

## 五、开始炒股前要知道

### 1. 模拟盘先练手

大多数券商APP都有模拟交易功能，先用模拟盘熟悉操作，亏了不心疼。

### 2. 第一笔不要太多

建议先用**1-2个月工资**的金额来练手，体验一下真实涨跌的心理感受。

### 3. 每天关注但不要每天操作

每天看一下大盘走势，熟悉市场节奏。但**不要频繁买卖**，手续费会吃掉你的收益。

## 六、券商APP基本功能

| 功能 | 说明 |
|------|------|
| 行情 | 看股票实时价格、涨跌幅 |
| 交易 | 买卖股票 |
| 持仓 | 看你买了哪些股票、盈亏多少 |
| 银证转账 | 银行卡和证券账户之间转钱 |
| 资讯 | 财经新闻、上市公司公告 |
