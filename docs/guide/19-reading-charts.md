# 第十九课：读盘技巧

选好了股票，什么时候买？看盘是关键技能。

## 一、盘面基础术语

| 术语 | 含义 |
|------|------|
| 集合竞价 | 9:15-9:25，试探当天开盘方向 |
| 开盘价 | 9:30第一笔成交价 |
| 收盘价 | 15:00最后一笔成交价 |
| 涨停价 | 当日最高能涨到的价格 |
| 跌停价 | 当日最低能跌到的价格 |
| 振幅 | 最高价到最低价的差距 |

## 二、看盘要看什么？

### 1. 看大盘环境

不是所有时候都适合买股票：

| 大盘环境 | 操作建议 |
|---------|---------|
| 大盘上涨 | 积极一些 |
| 大盘震荡 | 控制仓位 |
| 大盘下跌 | 谨慎观望 |

### 2. 看板块联动

个股要跟随板块走势：

- 板块整体涨 → 个股更容易涨
- 板块整体跌 → 个股难以独善其身

### 3. 看分时走势

| 分时特征 | 信号 |
|---------|------|
| 白线在黄线上方 | 权重股在拉指数 |
| 白线在黄线下方 | 小盘股更活跃 |
| 股价全天横着走 | 主力控盘，不上不下 |
| 股价突然急拉 | 小心是诱多 |

## 三、尾盘操作技巧

尾盘（14:30-15:00）是**最重要的操作时段**：

| 时间 | 适合做什么 |
|------|----------|
| 早盘（9:30-10:00） | 观察为主，不急着买 |
| 中盘（10:00-14:00） | 确认方向后操作 |
| 尾盘（14:30-15:00） | 最佳买入/卖出时机 |

> 💡 **老手多在尾盘买**：经过一天的博弈，价格更真实。

## 四、盘口语言

### 五档买卖盘

| 位置 | 含义 |
|------|------|
| 卖1-5 | 准备卖出的挂单 |
| 买1-5 | 准备买入的挂单 |
| 外盘（红） | 主动买入的成交量 |
| 内盘（绿） | 主动卖出的成交量 |

**外盘 > 内盘**：买方更积极，股票偏强
**内盘 > 外盘**：卖方更积极，股票偏弱

### 大单解读

| 现象 | 含义 |
|------|------|
| 大单买入 | 主力可能在吸筹 |
| 大单卖出 | 主力可能在出货 |
| 大单砸盘 | 可能有利空 |

## 五、集合竞价看盘

每天9:15-9:25是观察主力意图的好时机：

1. **看竞价涨幅**：大幅高开（>5%）要小心
2. **看成交量**：量太大可能是诱多
3. **9:20后不能撤单**：这时候的挂单更真实

> 🚨 **9:15-9:20挂涨停买入的，可能是庄家钓鱼**，别上当！

## 六、复盘的重要性

每天收盘后，花10分钟复盘：

| 复盘内容 | 目的 |
|---------|------|
| 今天赚钱了吗 | 总结操作 |
| 明天怎么看 | 制定计划 |
| 持仓股有没有问题 | 决定去留 |

## 思考题

> 今天你关注的某只股票，集合竞价时大幅高开7%，但9:20后成交量萎缩。你会追吗？为什么？
