# 前言

欢迎来到 **A股学习站**！这是一个为零基础投资者打造的免费学习平台。

## 为什么写这个教程？

A股市场有2亿开户账户，但真正理解股票投资的人少之又少。大多数人听说牛市来了就冲进去，结果成了"韭菜"。

这本书的目标是：**让一个完全不懂股票的小白，学完之后能够独立看懂盘面、选出自已想投资的股票**。

## 学习路径

教程共20课，分为8个章节：

| 章节 | 内容 | 课时 |
|------|------|------|
| 入门基础 | 股票是什么、开户、交易规则 | 3课 |
| 看懂市场 | 指数、板块、盘面 | 3课 |
| 基本面分析 | 财报、估值指标 | 2课 |
| 技术分析 | K线、均线、MACD | 3课 |
| 投资理念 | 价值投资、行业轮动 | 2课 |
| 基金入门 | ETF、定投 | 2课 |
| 风险控制 | 仓位、止损 | 2课 |
| 实战技巧 | 选股、读盘、下单 | 3课 |

## 学习建议

1. **按顺序学习**：每课都是建立在上一课的基础上
2. **动手实践**：学完立刻去券商APP模拟操作一遍
3. **控制仓位**：刚开始用很少的钱，亏了不心疼那种
4. **保持耐心**：股市不是提款机，需要长期学习积累
