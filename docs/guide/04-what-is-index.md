# 第四课：什么是指数？

炒股的人天天说"大盘今天涨了"、"创业板指创新高"，这个"指数"到底是什么？

## 一句话理解指数

**指数就是市场的"平均成绩单"。**

把很多只股票按一定规则打包在一起，算出一个综合分数，这个分数就是指数。

## 生活比喻

想象你们班有50个同学，期末考了5门课：
- **上证指数** = 全班所有人的平均分（所有股票的综合表现）
- **沪深300** = 全班成绩最好的30个人的平均分（最优秀的300只股票）
- **创业板指** = 艺术特长班的平均分（创新成长型公司）

## A股主要指数一览

### 上证指数（000001）

**全称：上海证券交易所综合股价指数**

- 包含了在上海证券交易所上市的**所有股票**
- 1991年发布，是中国历史最悠久的指数
- 你在新闻里听到"大盘跌破3000点"，说的就是它

> 📊 **上证指数 ≈ A股整体温度计**

### 深证指数（399001）

**全称：深圳证券交易所成分股价指数**

- 包含了在深圳证券交易所上市的**所有股票**
- 深圳市场以**中小盘、创业板**为主
- 整体风格比上证指数更"激进"

### 沪深300（000300 / 399300）

**最最重要的指数，没有之一。**

- 上海+深圳两家交易所里，**市值最大、流动性最好**的300只股票
- 相当于"全校前300名尖子生"
- A股最具代表性的核心资产

### 创业板指（399006）

- 专门跟踪**创业板**100只最大市值股票
- 创业板主要是：**创新型、成长型公司**（新能源、医药、科技为主）
- 波动比主板大，机会多、风险也高

> 🚨 创业板股票涨跌幅是±20%，比主板风险更大。

### 科创50（000688）

- 科创板（2019年新开）里**市值最大的50只股票**
- 科创板定位：**硬科技**企业（芯片、人工智能、生物医药等）
- 代表中国科技未来方向

## 指数的计算原理

不需要深究，但要知道：**指数是加权平均**。

大市值公司对指数影响更大（权重更高）。

### 举例

假设一个指数只有2只股票：
- 苹果公司市值1000亿，权重90%
- 小公司市值100亿，权重10%

如果苹果涨10%，指数大概涨9%；如果小公司涨10%，指数只涨1%。

> 所以常说"大盘涨了，但我的股票没涨"——很可能你买的是市值小的股票，不在权重股里。

## 为什么指数很重要？

### 1. 看大盘操作

新手最好**大盘环境好的时候才操作**，单边下跌的熊市减少交易。

| 大盘环境 | 操作建议 |
|---------|---------|
| 大盘连续上涨 | 可以积极一点 |
| 大盘震荡 | 控制仓位，少动 |
| 大盘连续下跌 | 多看少动，不抄底 |

### 2. 判断市场温度

| 指数点位 | 市场状态 |
|---------|---------|
| 上证 > 3500 | 偏热 |
| 上证 3000-3500 | 正常 |
| 上证 < 3000 | 偏冷（历史上在这个区间机会多） |
