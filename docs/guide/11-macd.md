# 第十一课：MACD指标

MACD是技术分析中最常用的指标之一，可以帮你判断趋势的力度和方向。

## 一、MACD是什么？

MACD = Moving Average Convergence Divergence（移动平均收敛发散指标）

**核心作用**：判断股票趋势是走强还是走弱。

## 二、MACD的构成

MACD由**三条线**和一个**红绿柱**组成：

| 组成部分 | 作用 |
|---------|------|
| DIF线（快线） | 短期与中期的指数均线差 |
| DEA线（慢线） | DIF线的均线，更平滑 |
| 红绿柱 | DIF与DEA的差值，表示力度 |

> 💡 你可以简单理解为：**红柱变长 = 上涨力度增强，绿柱变长 = 下跌力度增强**

## 三、MACD的核心用法

### 1. 金叉与死叉

**金叉（买入）**：DIF线上穿DEA线

**死叉（卖出）**：DIF线下穿DEA线

> 💡 和均线的金叉死叉类似，但MACD更平滑，信号更可靠。

### 2. 零轴

零轴是一条**分界线**：

| 位置 | 含义 |
|------|------|
| DIF > 0（零轴上方） | 短期均线在长期均线上方，偏强 |
| DIF < 0（零轴下方） | 短期均线在长期均线下方，偏弱 |

**操作原则**：
- 只做零轴上方的股票（只做强势）
- 零轴下方的金叉，可能是弱反弹，不参与

### 3. 背离（最重要！）

**顶背离**：股价创新高，但MACD没有创新高 → 预警！可能要跌

**底背离**：股价创新低，但MACD没有创新低 → 预警！可能要涨

> 💡 **背离是MACD最有价值的信号**，但要等走出来才能确认，不能预测。

## 四、MACD的局限性

> ⚠️ **MACD是趋势指标，在震荡市里容易反复打脸。**

- 横盘震荡时，MACD会频繁在零轴附近交叉
- MACD有滞后性，往往在行情走了一段后才出现信号
- **不能单独用MACD**，必须结合均线、K线形态、成交量

## 五、MACD实战口诀

```
零轴上方金叉，强势
零轴下方金叉，弱势（可能只是反弹）
高位死叉，赶紧跑
低位金叉，慢慢买
顶背离要小心
底背离要留意
```

## 思考题

> 某股票股价创了新高，但MACD没有创新高，形成了顶背离。你会怎么做？
