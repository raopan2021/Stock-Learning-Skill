# 第七课：读懂财报三张表

买股票就是买公司。判断一家公司好不好，最权威的资料就是**财报**。

## 一、财报从哪看？

| 渠道 | 地址 |
|------|------|
| 巨潮资讯网 | cninfo.com.cn（官方最全） |
| 东方财富 | eastmoney.com |
| 同花顺 | 10jqka.com.cn |
| 雪球 | xueqiu.com |

每个上市公司每年发布**4次财报**：
- 一季报（4月）
- 半年报（8月）
- 三季报（10月）
- 年报（次年4月前）

## 二、资产负债表

### 一句话理解

**公司有多少家当，欠了多少债。**

### 生活比喻

就像你的个人资产清单：
- 你有100万存款（资产）
- 欠银行80万房贷（负债）
- 净资产20万

### 关键科目

| 科目 | 含义 |
|------|------|
| 总资产 | 公司所有家当加起来 |
| 总负债 | 公司欠的所有债 |
| 净资产（所有者权益） | 总资产 - 总负债 |

### 判断要点

> 净资产 > 总负债的50%，说明公司比较稳健。

## 三、利润表

### 一句话理解

**公司赚钱了吗？赚了多少？**

### 生活比喻

就像月收入账单：
- 工资收入2万（营业收入）
- 扣除生活开销1.5万（营业成本）
- 净赚5000（净利润）

### 关键科目

| 科目 | 含义 |
|------|------|
| 营业收入 | 卖产品/服务收到的钱 |
| 营业成本 | 生产和销售过程的花费 |
| 净利润 | 扣除所有费用后真正赚到的 |

### 判断要点

> 净利润连续3年增长的公司，说明基本面不错。

## 四、现金流量表

### 一句话理解

**公司实际收到了多少现金？**

### 生活比喻

你月入2万，但其中1万是年底奖金（还没到账），实际本月到手只有1万。现金流量表看的是"真金白银"。

### 关键科目

| 活动 | 说明 |
|------|------|
| 经营活动现金流 | 主营业务真实收到的钱 |
| 投资活动现金流 | 买设备、并购等投资花的钱 |
| 筹资活动现金流 | 借钱、发行股票等融资行为 |

### 判断要点

> **经营现金流为负**的公司要小心，说明主业不赚钱，靠融资活着。

## 五、财务最重要的原则

> 💡 **利润可以"做账"，但现金流量表很难造假。**

所以专业投资者最关注的指标：**经营现金流 > 净利润**

如果一家公司净利润很高，但经营现金流长期为负，就要怀疑利润的真实性。

## 六、简化版财务筛选

用东方财富的选股器，可以快速筛选：

| 条件 | 建议值 |
|------|--------|
| 净利润增速 | >10% |
| 净资产收益率(ROE) | >15% |
| 资产负债率 | <60% |
| 市盈率(PE) | <30（成长股可接受更高） |

## 七、三张表的关系

```
资产负债表（某一时点的照片）
    ↓
利润表（一段时间的录像）
    ↓
现金流量表（录像里真金白银的部分）
```

三者要**结合看**，缺一不可。

## 思考题

> 某公司净利润很高，但经营现金流连续2年为负。可能是什么原因？这种公司能买吗？
